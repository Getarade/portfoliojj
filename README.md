# The Guild - Web Development Portfolio

A modern, responsive portfolio website for The Guild web development team. This site showcases the team's services, expertise, and contact information.

## Features

- Clean, modern design with dark theme
- Responsive layout for all devices
- Animated sections and transitions
- Service showcase
- Team information
- Contact details

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts

## Live Demo

Visit the live site: [The Guild Portfolio](https://getarade.github.io/portfoliojj/)

## Setup

Simply clone this repository and open the index.html file in your browser to view the site locally.

```
git clone https://github.com/getarade/portfoliojj.git
cd portfoliojj
```
